New block sequence

 - Block gets added
 - Grid triggers onBlockAdd event.
 - Scene catches event and creates new sprite and connector.
 - Add sprite to grid.
 - Add connector to array.

Block explosion sequence

 - Grid explodes block.
 - Block triggers explosion event.
 - Connector catches explosion event.
 - Connector runs animation.
 - Animation ends.
 - Connector calls block's stopExploding method.
 - Block triggers explosion stopped event.
 - Connector catches explosion stopped event and removes sprite/marks block as
   dead.
 - Block is removed from grid.
 - Connector is removed from array.

Block movement sequence

 - Grid moves block.
 - Block triggers movement event.
 - Connector catches movement event.
 - Connector moves changes sprite co-ords to match the block.

Block landing sequence

 - Grid tells block it has landed.
 - Block triggers landing started event.
 - Connector catches landing started event.
 - Connector runs animation.
 - Animation ends.
 - Connector calls block's stopLanding method.

 - Grid triggers landing event.
 - Layer catches landing event.
 - Layer triggers landing sound.
